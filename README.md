# analisis-videojuegos-ice-store
Análisis de datos de videojuegos para Ice Store. Identificación de patrones de éxito en ventas globales, análisis por plataformas, géneros y regiones (NA, EU, JP). Incluye preprocesamiento, EDA, pruebas de hipótesis y perfiles de usuario regionales para optimizar campañas publicitarias 2017.
